## Project Overview

Amzad Food ERP is designed to manage end-to-end operations for a food manufacturing business. This admin panel allows staff and managers to monitor inventory, production plans, packaging, transfers, sales, returns, and generate reports.

Key modules include:

- Authentication (Login, Forgot Password, Reset Password)
- Master Setup (Units, Brands, Raw Materials, Packaging, Products, Combo Products, BOM)
- Inventory & Store Management (Purchase, Stock, Requisitions)
- Production Management (Planning, Execution)
- Packaging & Transfer (Requisition, Execution, QR-based transfers)
- Sales & POS
- Reporting (Stock, Production Cost, Sales)
- Users, Roles, and Permissions Management

---

## Features

- Fully responsive **dashboard** with charts and analytics.
- **CRUD operations** for master data and inventory items.
- **Production workflow** with BOM-based material calculation.
- **POS system** with cart, QR scanning, and payment processing.
- **Reports** with charts and filters.
- User management with **roles and permissions**.
- QR code support for packaged products and transfers.
- Optimized for **performance**, **SEO**, and **user experience**.

## Technology Stack

- **Frontend:** React, TypeScript, Tailwind CSS, shadcn/ui, React Hook Form
- **State Management:** Redux, RTK Query
- **Charts:** Recharts
- **Forms & Tables:** Ant Design
- **Utilities:** clsx, class-variance-authority
- **Icons:** Lucide React, React Icons

## Installation

Clone the repository:

```bash
git clone https://github.com/ibnabdullah1/admin.amzadfood.git
cd admin.amzadfood
```

Install dependencies:

```bash
yarn install
# or npm install
```

---

## Running the Project

Start the development server:

```bash
yarn dev
# or npm run dev
```

Build the project for production:

```bash
yarn build
# or npm run build
```

Preview production build:

```bash
yarn preview
# or npm run preview
```
